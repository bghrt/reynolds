If we don't have it, you can still get it!

Want to double check if we have it? Check QuickSearch if you're looking for a book or DVD. Check the Journal Locator if you know the name of the journal. The Journal Locator will tell you if the journal is available online or in print through Reynolds Library.

If Reynolds Library doesn't have the item, students, faculty, and staff can choose to travel to a local library and check out the item or request it online via the Interlibrary Loan Service (ILL).

To Check Out an Item Directly From a Local Library

The Cooperative Borrowing Program allows current Reynolds students, faculty, and staff to borrow library materials directly from participating academic libraries such as the University of Richmond and Virginia Commonwealth University.

The RALC (Richmond Academic Library Consortium) allows students to visit John Tyler, Randolph Macon, or Union Presbyterian and check out up to five books from their collection. Ask at any library reference desk for a RALC pass.

To find out if a local library owns an item, check WorldCat or the local lending library;s website. Here is a list of some local public libraries and their websites:

Chesterfield County Library: http://library.chesterfield.gov/

County of Henrico Public Library: http://www.henricolibrary.org/

Jefferson-Madison Public Library (Louisa): http://jmrl.org/

Library of Virginia: http://www.lva.virginia.gov/

Pamunkey Regional Library: http://www.pamunkeylibrary.org/

Powhatan Public Library: http://public.powhatanlibrary.net

Richmond Public Library: http://www.richmondpubliclibrary.org/

To Request an Item Online Through Interlibrary Loan

If you don't want to travel to a local library, we can interlibrary loan it for you! Just fill out the Interlibrary Loan Form.

Questions or need more information? Ask any librarian for help.
