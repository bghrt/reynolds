Purpose / Bias

  • What is the purpose of the source? Is it to inform, persuade, present opinions, report research, or sell a product? For what audience is the source intended?
Types of Web Sites
  • commercial
  • educational
  • entertainment
  • hoax
  • informative
  • organizational
  • persuasive
  • personal
  • Does the source show any bias? What opinions (if any) are expressed by the author? If there is an issue, are both sides presented or is it ones-sided?
  • Is the source popular or scholarly?

How can I find out?

  1. Read the source you are evaluating. Look for clues in the text and illustrations.
  2. Determine whether the source is published by an organization with a particular purpose.
  3. If it is a web site, look for links to pages that tell more about the content of the Web site such as About this Site.
  4. Determine whether the source attempts to sell a product or promote a particular point of view. Also, see if it presents a balanced view.
  5. Look for any advertising.
  6. Determine whether the material is scholarly or popular in nature by using the following criteria:
